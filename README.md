# AI-based models trained on site flux measurements reveal the spatiotemporal dynamics of daily cropland N2O emissions across Europe

This repository contains the visualization scripts associated with the manuscript:

Li et al.
"AI-based models trained on site flux measurements reveal the spatiotemporal dynamics of daily cropland N2O emissions across Europe."


## System requirements

The scripts have been tested on:
  Windows 11
  
Software:
  MATLAB R2024b

Required toolboxes/packages:
  MATLAB Mapping Toolbox 
  Statistics and Machine Learning Toolbox 24.2

## Installation

Clone the repository:
  git clone https://github.com/zimengli-PKU/EU_N2O/

Open MATLAB.

Add the repository to the MATLAB path.

No additional installation is required.

## Demo

Source data that supported the figures are provided in the Data folder.

Drafting the figures only take several seconds.

## Instructions for use

Please 'load *.mat' before running the codes.
